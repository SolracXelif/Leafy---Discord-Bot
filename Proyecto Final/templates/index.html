Bienvenido a Leafy

Tu asistente personal para el clima y la lucha contra el cambio climático

Obtén información del clima en tiempo real, consejos ecológicos prácticos y noticias sobre el cambio climático, todo en una plataforma interactiva.

Funciones Principales

Información del Clima

Consulta el clima actual y los pronósticos detallados para tu ubicación. Accede a datos meteorológicos en tiempo real y prepárate para cualquier condición climática.

Consejos Ecológicos

Recibe consejos prácticos y accionables para reducir tu impacto ambiental. Aprende cómo contribuir a la protección del planeta desde tu día a día.

Noticias Climáticas

Mantente informado con las últimas noticias relacionadas con el cambio climático. Conoce los avances científicos y políticas de acción climática global.

Tecnología Utilizada

Librerías de Python

Leafy está construido con las siguientes librerías de Python:

discord requests pyttsx3
Discord

Librería oficial para crear bots de Discord. Permite interactuar con la API de Discord y responder a eventos de servidor.

Requests

Cliente HTTP para Python. Se utiliza para consumir APIs de información meteorológica y datos climáticos.

Pyttsx3

Motor de síntesis de voz texto-a-voz. Proporciona información climática en formato de audio.

Cómo Usar Leafy

Habla con Leafy!

Leafy es un bot de Discord con una personalidad única que utiliza emojis, imágenes y síntesis de voz para informarte sobre:

  • Cambios climáticos y sus consecuencias
  • Soluciones sostenibles y ecológicas
  • Datos relacionados con el clima global
  • Noticias del mundo sobre cambio climático

Es una guía interactiva para la concienciación sobre el cambio climático en tu servidor de Discord.

Sobre Leafy

Leafy es un bot de Discord con una personalidad única dedicado a la concientización sobre el cambio climático. Su objetivo es:

  • Informar sobre el cambio climático y sus impactos globales
  • Proporcionar datos científicos confiables y actualizados
  • Ofrecer soluciones prácticas y accesibles para cada persona
  • Crear una comunidad consciente del ambiente en Discord
  • Inspirar acciones positivas para proteger nuestro planeta
